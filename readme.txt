Realm Status Box für ilchClan 1.1:
----------------------------------

Beschreibung:
-------------
Dieses Box-Modul zeigt euch in einer Box ob der Realm Server Online oder Offline ist.

Entwickelt
----------
° von "Siggi"
 
Installation:
-------------
° alle Dateien im Ordner upload, in ihrer Ordnerstruktur hochladen
  !! Vorsicht: Überschreibt die: !!
	| include/admin/allg.php
° Nach uploaden aller Datein muss die realmstatus_install.php ausgeführt werden.
    => index.php?realmstatus_install
° Im Adminbereich unter Navigation die Box realmstatus.php einbinden.
° Anschließend im Adminbereich unter Konfiguration "World of Warcraft Optionen" Angaben eurer Gilde anpassen.
